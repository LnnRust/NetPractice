# NetPractice

## Description

NetPractice is a practical networking exercise from the 42 curriculum. Through 10 progressive levels, you are given broken network diagrams and must configure them correctly so all devices can communicate. The project covers TCP/IP addressing, subnetting, routing, and network topology — no coding required.

## Instructions

Open `index.html` in your browser, or run:

```bash
bash run.sh
```

Solve each level, then export your configuration with the **Get my config** button. Place the 10 exported `.json` files (one per level) at the root of your repository before submitting.
